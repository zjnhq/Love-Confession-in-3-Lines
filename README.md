# Pennsylvania Recollection

Preface:  <br />
不是曹雪芹， <br />
写不出红楼梦， <br />
做不到文案的每一个字，每一个标点， <br />
都精心构思认识路上的点点滴滴， <br />
你的名字和联系方式， <br />
温馨的聊天记录， <br />
满满的泪和喜欢。 <br />

始于费城也坠于费城， <br />
没能飞到拉斯维加斯， <br />
那只是沙漠里的海市蜃楼。 <br />
